## Contexte du Projet
Ce projet a été réalisé dans le cadre de l'examen **"Mini pipeline de bout en bout"**.

L'objectif est de développer une solution de **Machine Learning (NLP)** capable de prédire automatiquement si un contenu Netflix (Film ou Série) est destiné à un **public adulte ("Mature")** ou à un **public général ("General")**, en analysant uniquement son résumé (*description*).

**Dataset :** [Netflix Movies and TV Shows (Kaggle)](https://www.kaggle.com/datasets/shivamb/netflix-shows)
